# 管理画面と管理者ログイン機能の実装

## 概要
Supabase Authを使用した管理者認証と、記事・コンテンツを管理できる管理画面を実装します。

## 実装ステップ

### 1. 環境変数の設定
- `.env.local` ファイルを作成し、Supabase認証情報を追加

### 2. Supabase認証の設定
- Supabaseクライアントの認証機能を有効化
- 認証用のヘルパー関数を作成

### 3. ログインページの作成
- `/admin/login` ページを作成
- メールアドレスとパスワードでログイン
- モダンで美しいUIデザイン

### 4. 認証ミドルウェアの実装
- ログイン状態をチェックする機能
- 未ログイン時はログインページへリダイレクト

### 5. 管理画面ダッシュボードの作成
- `/admin` ページを作成
- 記事一覧、統計情報を表示
- ナビゲーションメニュー

### 6. 記事管理機能
- `/admin/posts` - 記事一覧
- `/admin/posts/new` - 新規記事作成
- `/admin/posts/[id]/edit` - 記事編集
- 記事の公開/非公開切り替え
- 記事削除機能

### 7. 画像アップロード機能
- Supabase Storageを使用した画像アップロード
- 画像プレビュー機能

### 8. ログアウト機能
- セッション管理
- ログアウトボタンの実装

## 技術スタック
- **認証**: Supabase Auth
- **データベース**: Supabase Database
- **ストレージ**: Supabase Storage
- **UI**: Tailwind CSS + Lucide Icons
- **フォーム**: React Hook Form (必要に応じて)

## セキュリティ考慮事項
- Row Level Security (RLS) の設定
- 管理者ロールの設定
- CSRF対策
- XSS対策
